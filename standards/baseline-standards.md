# Baseline Standards

## Purpose
Define enterprise standards for Intune security and compliance baselines.

## Naming Convention
- CFG - Windows - Device Restrictions - v1.0
- CMP - Windows - Secure Baseline - v1.0
- SEC - Defender - AV + ASR Baseline - v1.0

## Deployment Rings
- Ring 0: IT / Test
- Ring 1: Pilot
- Ring 2: Department
- Ring 3: Enterprise

## Change Control
Every baseline change must document:
- What changed
- Why it changed
- Risk
- Rollback plan
- Validation steps

## Documentation Requirements
Each baseline must include:
- Objective
- Scope
- Key settings
- Validation steps
- Troubleshooting notes
